# gitflow
work with version control tool git
远端初始化仓库
远端仓库clone到本地
开始编写代码，本处编写readme
即将进行第一次文件添加，check status 验证更改
进行第一次commit 并最佳提交描述信息
完成本次代码编写存储，可以推送到远端

新建分支开发独立功能，此内容只应出现在分支lch内
进行内容变更的储存，提交，推送

分支lch 内容应出现在main分支，lch分支不在存在
